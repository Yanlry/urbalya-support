Politique de confidentialité

Urbalya collecte uniquement les informations nécessaires au fonctionnement de l’application (compte, contenus, notifications et, si vous l’autorisez, localisation). Aucun usage publicitaire sans consentement.

Dernière mise à jour : 25 janvier 2026

1) Qui sommes-nous ?

Urbalya est une application mobile permettant de consulter l’actualité locale, publier des signalements, échanger et visualiser une carte de la ville.

Contact (confidentialité / RGPD) : contact@urbalya.com

2) Données que nous traitons

  • Compte : e-mail, pseudo, mot de passe (hashé côté serveur), photo de profil (facultatif).
  • Contenus : signalements, commentaires, photos, pièces jointes.
  • Localisation : position précise/approximative lorsque vous l’autorisez (ex : carte, proximité).
  • Identifiants techniques : identifiant interne, jeton de notification push, logs de sécurité.
  • Diagnostics : erreurs et performances (si activés / nécessaires au support).
Information : aucune donnée bancaire n’est stockée par Urbalya.

3) Finalités et bases légales

  • Fournir le service (compte, publications, carte) : exécution du service.
  • Sécurité, prévention des abus, journalisation : intérêt légitime.
  • Notifications : intérêt légitime et/ou consentement (push).
  • Localisation : consentement (désactivable dans iOS).
  • Amélioration produit (diagnostics anonymisés) : intérêt légitime.
Important : vous pouvez retirer votre consentement (localisation / notifications) à tout moment via les réglages iOS.

4) Destinataires et transferts

  • Hébergement : serveur applicatif et base de données.
  • Stockage médias : Amazon Web Services (ex : S3).
  • Notifications : services push (ex : Expo / Apple / Google).

Nous ne vendons pas vos données. Aucun suivi publicitaire inter-apps.

5) Durées de conservation

  • Compte et contenus : jusqu’à suppression du compte ou demande d’effacement (selon obligations légales).
  • Logs techniques : jusqu’à 12 mois (sécurité/support).
  • Sauvegardes : cycles techniques (jusqu’à 30 jours).

6) Vos choix et droits (RGPD)

Réglages iOS :

  • Localisation : iOS → Réglages → Urbalya → Localisation → “Jamais / Quand l’app est active”.
  • Notifications : iOS → Réglages → Notifications → Urbalya.
  • Suppression de compte : dans l’app → Paramètres → Compte → Supprimer mon compte.

Droits : accès, rectification, effacement, limitation, portabilité, opposition, retrait du consentement. Pour exercer vos droits : contact@urbalya.com.

7) Sécurité

Chiffrement en transit (HTTPS), accès restreints, pratiques de sécurité (mots de passe hashés, contrôles d’accès, journalisation).

8) Mineurs

L’app ne vise pas les mineurs sans autorisation parentale lorsque la loi l’exige.

9) Modifications

Nous mettrons à jour cette page en cas de changement substantiel, avec une date de mise à jour visible en haut du document.
