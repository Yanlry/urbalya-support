FAQ & solutions

Recherche instantanée + réponses actionnables.
Entrée
Je ne reçois pas les notifications
  • iOS → Réglages → Notifications → Urbalya → activer Autoriser.
  • Android → Paramètres → Applications → Urbalya → Notifications → activer Autoriser.
  • Vérifier les modes Concentration / Ne pas déranger.
  • Ouvrir l’app au moins une fois après installation (initialisation des permissions).
La localisation ne fonctionne pas / pas de précision
  • iOS → Réglages → Confidentialité → Service de localisation → Urbalya → Lorsque l’app est active.
  • Android → Paramètres → Applications → Urbalya → Autorisations → activer Localisation (pendant l’utilisation).
  • Activer la précision de localisation pour de meilleurs résultats sur la carte.
  • Sans localisation, certaines fonctions sont moins précises (affichages par ville, etc.).
Comment supprimer mon compte ?
  • Dans l’app : Paramètres → Compte → Supprimer mon compte.
  • Via le web : page publique de suppression de compte.
  • Pour une demande RGPD spécifique : contact@urbalya.com.
L’app crash / écran blanc / bug bloquant
  • Redémarrer l’app et vérifier la connexion réseau.
  • Mettre iOS ou Android à jour et installer la dernière version de Urbalya.
  • Envoyer un rapport : modèle d’appareil + plateforme + version OS + capture + heure + étapes à contact@urbalya.com.
Contenu choquant / harcèlement : que faire ?
  • Dans l’app : utilisez Signaler + Bloquer.
  • Par email : contact@urbalya.com avec captures, ID/lien, description.
