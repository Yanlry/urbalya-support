Demande de suppression de compte

Vous pouvez supprimer votre compte directement dans l’application ( Paramètres → Compte → Supprimer mon compte ) ou via cette page publique. Cette page ne vous demandera jamais votre mot de passe.

Après confirmation par lien/code, la suppression est traitée immédiatement côté compte. Certaines traces techniques ou de sécurité peuvent rester temporairement dans les sauvegardes selon les obligations légales.

1) Recevoir un lien de confirmation

Entrez l’email du compte. Nous envoyons un lien/code temporaire (expiration courte).

2) Confirmer la suppression

Ouvrez le lien reçu par email, ou collez ici le code temporaire à usage unique.

Ce qui est supprimé et ce qui peut être conservé temporairement

  • Supprimé : profil utilisateur, accès au compte, contenus liés (selon règles de cascade applicatives).
  • Conservation temporaire possible : logs de sécurité, traces anti-abus/modération, sauvegardes techniques.
  • Motifs : obligations légales, sécurité, prévention de fraude, gestion des litiges/modération.
  • Délai : traitement principal immédiat après confirmation ; purge complète des sauvegardes selon cycle technique (jusqu’à 30 jours).

Pour une demande RGPD spécifique : contact@urbalya.com.
